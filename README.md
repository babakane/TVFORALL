## 🚀 Getting Started

### Quick Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

### Access the App

- **Development**: [http://localhost:3000](http://localhost:3000)
- **GitHub Pages**: [https://babakane.github.io/TVFORALL](https://babakane.github.io/TVFORALL)
