# TVFORALL ⚽🌍

A comprehensive platform providing **live channels** for football and **World Cup** coverage with real-time streaming availability.

## 🎯 Overview

TVFORALL is a modern web application built with **Next.js** and **TypeScript** that aggregates and displays live football channels and World Cup events with all available streaming options.

## ✨ Features

- 📺 **Live Football Channels** - Access to major football broadcasting channels
- 🏆 **World Cup Coverage** - Complete World Cup event streaming
- 🌐 **Multi-Channel Support** - View all available streaming options
- 🎨 **Modern UI/UX** - Responsive and intuitive interface
- ⚡ **Fast & Reliable** - Built with Next.js for optimal performance

## 🛠️ Tech Stack

- **Frontend**: Next.js, TypeScript, React
- **Styling**: PostCSS, Tailwind CSS
- **Package Manager**: pnpm
- **Build Tool**: Next.js

## 📦 Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## 📁 Project Structure

```
├── app/              # Next.js app directory
├── components/       # Reusable React components
├── hooks/           # Custom React hooks
├── lib/             # Utility functions and helpers
├── styles/          # Global styles
├── public/          # Static assets
├── timetv.html      # Main HTML page for live channels
└── package.json     # Project dependencies
```

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Start the development server: `pnpm dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🌟 Key Files

- **timetv.html** - Primary interface for accessing live channels and World Cup streams
- **components.json** - Component configuration
- **next.config.mjs** - Next.js configuration
